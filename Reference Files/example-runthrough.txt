Example Runthrough (LOCAL mode)

Purpose: demonstrate a *successful* local-folder extraction run using `sample.rtf`.
This run reflects the current RTF timeframe parsing behavior (so you should not see
spurious “No timeframe specified” warnings for rows that contain time ranges).

How to reproduce (high level):
1. Run: `./run_clip_extractor.sh`
2. Choose mode: `D` (LOCAL folder extraction)
3. Select RTF: `sample.rtf`
4. Choose a destination folder for outputs (folder picker)
5. Buffer mode: enter `2` seconds when prompted (2s before + 2s after)
