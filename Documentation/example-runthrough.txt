🍪 YOUTUBE COOKIE PROFILES SETUP
============================================================
To bypass YouTube's quality restrictions, you can create multiple
cookie profiles. The script will try each one until it finds one
that allows 720p+ downloads.

📁 Cookie Files to Create:
   • youtube_cookies.txt           (Primary profile)
   • youtube_cookies_profile2.txt   (Secondary profile)
   • youtube_cookies_profile3.txt   (Tertiary profile)
   • youtube_cookies_private.txt    (Private browser profile)

🔧 How to Create Cookie Files:
1. Install a browser extension like 'Get cookies.txt' or 'cookies.txt'
2. Go to YouTube.com and log in with different accounts
3. Use the extension to export cookies
4. Save as the appropriate filename in the script directory

🌐 Alternative: Use Private Browser Session
1. Open YouTube in private/incognito mode
2. Export cookies from private session
3. Save as 'youtube_cookies_private.txt'

💡 Pro Tips:
   • Different accounts may have different quality access
   • Premium accounts often get better quality
   • Private sessions sometimes bypass restrictions
   • The script will try all available profiles automatically
